Pregunta 2.

¿Que es un Framework CSS?

Un framework o marco de trabajo traducido al español es una capa de desarrollo inicial que nos ofrece un inicio no completamente desde cero sino una base ya creada. Existen frameworks para todos los lenguajes, desde Java y Spring hasta PHP y Laravel y CSS no es menos

Un framework de CSS no es más que una librería con una serie de estilos ya prediseñados que suele venir con algún tipo de compilador como lo es Tailwind CSS u otros ya precompilados como Bootstrap.

¿Que es Bootstrap?

Bootstrap es un framework CSS precompilado (Unicamente hay que descargar el fichero CSS y no hay que instalar ningún software adicional) que ofrece una forma rápida y eficiente de diseñar aplicaciones responsives gracias al layout basado en columnas que ofrece

¿Que es Tailwind CSS?

TailWind CSS al igual que bootstrap es un framework CSS que tiene como objetivo ofrecer una manera facil rápida y sencilla de diseñar páginas web con diferentes estilos ya incorporados aunque es necesario compilarlo debido a que TailWind CSS elimina las clases que no se estén utilizando en el proyecto una vez entrado en producción para así tener archivos CSS más livianos y eficientes

¿Cual es la diferencia entre TailWind CSS y Bootstrap

Las diferencias son las siguientes
  Bootstrap TailWind CSS
Precio Completamente gratuito ya que es un proyecto código abierto Su precio es 0€ para uso como hobby aunque si se quiere utilizar todo su potencial habrá que comprar la licencia personal de 249€ o la empresarial de 749€
Eficiencia Es un archivo completo con +20.000 lineas lo que puede ralentizar antiguos equipos Gracias a su compilador, solo se dejan las clases que se utilizan obteniendo así un archivo mucho más eficiente que bootstrap
Comunidad Tiene una comunidad a nivel mundial ya que es uno de los frameworks más utilizados por lo que hay multitud de plantillas y soporte No es tan conocido como Bootstrap por lo que es posible que no tenga algunos componentes, plantillas o incluso la misma cantidad de dudas resueltas que su contrincante
