Terminos

¿Que es un servidor Web?

Un servidor web es un servicio el cual se encarga de escuchar peticiones provenientes de los puertos 80 y 8080 para el protocolo HTTP y 443 para el protocolo HTTPS. Su deber es analizar la petición, usualmente una petición GET seguida con los headers y devolver un resultado en consencuencia si el recurso que está solicitando es encontrado enviará dicho contenido y un código 200, si por ejemplo en su contrario no es encontrado se enviará el código de error 404. Hay muchos servidores web pero los más famosos son Apache, Nginx y Tomcat aunque hay más con menos relevancia como lighttpd

¿Que es el protocolo HTTP?

El protocolo HTTP o Hyper-Text Transfer Protocol (Protocolo de transferencia de hipertexto) es un protocolo creado en el 1990 (Primer servidor creado) principalmente pensado para el intercambio de documentos cientificos en el CERN aunque con el paso del tiempo se ha convertido en uno de los protocolos mas usado a nivel mundial.

HTTP es un protocolo que consta de una serie de verbos, usualmente llamados verbos HTTP que permiten al usuario realizar diferentes tipos de consultas a un servidor web. El verbo más estandar que cualquier usuario común usuria sería el verbo GET pues es el verbo encargado de solicitar el contenido de la página web al servidor web

Aunque también está el verbo POST que inconscientemente usan cuando envían formularios de cualquier tipo por ejemplo, de registro.

Pero existen más verbos que no se suelen utilizar normalmente, y esos son los siguientes

  • DEL
  • PUT
  • PATCH

¿Que es Apache?

Apache es una compañia centrada en la creacion de software libre y dueña de el servidor Web más utilizado llamado Apache Web Server, esto es así gracias a las multiples ventajas que ofrece como ser gratuito y código abierto, tener una gran comunidad, ser ligero y ofrecer opciones como por ejemplo poder establecer host virtuales.

¿Que es XAMPP?

XAMPP es lo que se denomina un "stack de soluciones" pues es un paquete que incluye varios servicios que pueden resultar esenciales a la hora de desarrollar una página web como por ejemplo MySQL.

XAMPP fue desarrollado por Apache Friends siendo multiplataforma y es otra alternativa como lo son WAMP y LAMP para Windows y Linux respectivamente.

El stack de soluciones de XAMPP trae varios servicios que podremos decidir si instalar o no cuando instalemos XAMPP entre ellos hay un servidor Web HTTP, un servidor MySQL o MariaDB dependiendo de la versión, PHP y Perl, aunque también ofrece un servidor FTP y varias otras herramientas
